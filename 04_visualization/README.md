# 04 — Data Visualization & Storytelling

Dataset: Breast Cancer Wisconsin  
A 4-panel dashboard combining class distribution, boxplots, scatter, and violin charts
into a cohesive analytical story — the kind of output used in stakeholder reporting.

| Technique | Chart Type |
|---|---|
| Distribution overview | Bar chart with data labels |
| Feature separation | Side-by-side boxplots |
| Two-feature relationship | Scatter with class color |
| Distribution shape | Violin plot |

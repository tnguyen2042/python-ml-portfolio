# 01 — Exploratory Data Analysis

Dataset: Breast Cancer Wisconsin (`sklearn.datasets.load_breast_cancer`)  
569 samples · 30 features · binary target (Malignant / Benign)

| Step | What you'll see |
|---|---|
| Shape, dtypes, `.describe()` | Basic profiling |
| Null audit | Completeness check |
| Class balance bar chart | Target distribution |
| Feature histograms by class | Separation between classes |
| Correlation heatmap | Multi-collinearity check |
| Target correlation bar chart | Most predictive features |
